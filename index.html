HTML-Spiel

Dieses Spiel ist ein Demo-Projekt für die Programmierung mit React; es wird ausführlich erklärt in einem Tutorial in c't Ausgabe 2/16.

Datengrundlage ist die W3C-Empfehlung HTML 5.1-2 (Oktober 2017); als experimentell hinzugefügt wurden die in WHATWG HTML (Stand: Mai 2018) zusätzlich verzeichneten Elemente.

Inspiriert vom HTML5 Quiz und einem älteren Quiz auf justSayHi.com, das mittlerweile aus dem Netz verschwunden ist.
